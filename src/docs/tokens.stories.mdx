# Design Tokens

Design tokens are all the values (e.g., color, typography, object styles, spacing, animation, etc.) needed to construct and maintain a design system represented as data/variables. These can represent anything defined by design: a color as an RGB value, an opacity as a number, an animation ease as Bezier coordinates. Theyâ€™re used in place of hard-coded values in order to ensure flexibility and unity across all product experiences.

The following types of design tokens are the building blocks that make up the DLP design language:
