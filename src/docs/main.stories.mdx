# Main Content

* Main content is divided into sections (e.g., Agent, Co-Agent, Team, etc.). 
* Each section is coded as `article` and nested inside `main`.
* The Contact section includes legal disclaimers. It is coded as `footer` and sits outside of `main`. See [Footer](/?path=/story/pattern-library-containers-footer--page).
* Some sections have a 1px light gray (Primary 200) divider between them (i.e., `.section-divider`). Specifically, Agent | Co-Agent | Team... Community | Footer. See [Section Content](/?path=/story/pattern-library-containers-section-content--page).
* Some sections have an intro (e.g., Company, Technology, Marketing); others do not (e.g., Co-Agent, Team, Contact). 
* Each intro is coded as a `header` for that `article`. See [Section Intro](/?path=/story/pattern-library-containers-section-intro--page).
* All sections have section content, which aligns to a grid. See [Section Content](/?path=/story/pattern-library-containers-section-content--page).
