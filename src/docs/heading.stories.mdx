# Heading

* Headings are dark gray (Primary 900), Gotham Light (300) or Medium (500), various sizes (see below), 150% leading (line-height), 5 or 10% tracking (letter-spacing). 
* **Styling**: [Accents](/?path=/story/style-guide-object-styles--page#accents) may be added to headings.
* **Fit text**: Level 1 headings on mobile will scale to fit the available width. They have `font-size: 5vh;` instead of a fixed font-size.
* **Spacing**: 24/32px between `.head-1` and `.head-5` on mobile/desktop.
* **Animation**: Headings can fade in. Opacity changes from 0 to 1 at the same speed as the user scrolls (scrub). Each `<hX>` is given the class `.fade-in`: `<h2 class="head-1 fade-in">` (*not* indicated in code example below). See [Animation](/?path=/docs/style-guide-animation--page).

## How to configure

Define classes in **src/assets/type.styles.scss**.

<Source language='scss' code={dedent`
  /* Heading */
  .head-1, .head-2, .head-3, .head-4, .head-5, .head-6 {
    @apply leading-normal;     /* line-height: 1.5em; */
    @apply uppercase;          /* text-transform: uppercase; */
  }
  .head-1, .head-2, .head-3, .head-4 {
    @apply tracking-wider;     /* letter-spacing: 0.05em; */
  }
  .head-5, .head-6 {
    @apply font-medium;        /* font-weight: 500; */
    @apply tracking-widest;    /* letter-spacing: 0.10em; */
  }
  .head-1 { font-size: 5vh }   /* font-size: 5vh */
  .head-2 { @apply text-3xl; } /* font-size: 34px */
  .head-3 { @apply text-2xl; } /* font-size: 28px */
  .head-4 { @apply text-xl; }  /* font-size: 24px */
  .head-5 { @apply text-lg; }  /* font-size: 20px */
  .head-6 { @apply text-sm; }  /* font-size: 14px */
  @media (min-width: theme('screens.lg')) {
    .head-1 { @apply text-6xl; }  /* font-size: 60px */
    .head-2 { @apply text-5xl; }  /* font-size: 42px */
    .head-3 { @apply text-4xl; }  /* font-size: 39px */
    .head-4 { @apply text-3xl; }  /* font-size: 34px */
    .head-5 { @apply text-xl; }   /* font-size: 24px */
    .head-6 { @apply text-base; } /* font-size: 16px */
  }
  /* Heading: spacing */
  .head-1 + .head-5 {
    @apply mt-6;               /* margin-top: 24px; */
    @media (min-width: theme('screens.lg')) {
      @apply mt-8;             /* margin-top: 32px; */
    }
  }
`} />
