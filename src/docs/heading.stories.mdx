# Heading

* Headings are dark gray (Primary 900), Gotham Light (300) or Medium (500), vary in size (see below), 150% leading (line-height), 5-10% tracking (letter-spacing). 
* [Accents](/?path=/story/style-guide-object-styles--page#accents) may be added to headings.
* See [Spacing](/?path=/docs/style-guide-spacing--page) for details about space around headings.

## How to configure

Define classes in **src/assets/type.styles.scss**.

<Source language='css' code={dedent`
  /* Heading */
  .head-1, .head-2, .head-3, .head-4, .head-5, .head-6 {
    @apply leading-normal;     /* line-height: 1.5em; */
    @apply uppercase;          /* text-transform: uppercase; */
  }
  .head-1, .head-2, .head-3, .head-4 {
    @apply tracking-wider;     /* letter-spacing: 0.05em; */
  }
  .head-5, .head-6 {
    @apply font-medium;        /* font-weight: 500; */
    @apply tracking-widest;    /* letter-spacing: 0.10em; */
  }
  .head-1 { @apply text-5xl; } /* font-size: 42px */
  .head-2 { @apply text-3xl; } /* font-size: 34px */
  .head-3 { @apply text-2xl; } /* font-size: 28px */
  .head-4 { @apply text-xl; }  /* font-size: 24px */
  .head-5 { @apply text-lg; }  /* font-size: 20px */
  .head-6 { @apply text-sm; }  /* font-size: 14px */
  @screen lg {
    .head-1 { @apply text-6xl; }  /* font-size: 60px */
    .head-2 { @apply text-5xl; }  /* font-size: 42px */
    .head-3 { @apply text-4xl; }  /* font-size: 39px */
    .head-4 { @apply text-3xl; }  /* font-size: 34px */
    .head-5 { @apply text-xl; }   /* font-size: 24px */
    .head-6 { @apply text-base; } /* font-size: 16px */
  }
  /* Heading: spacing */
  .head-1 + .head-5 {
    @apply mt-6;               /* margin-top: 24px; */
    @screen lg {
      @apply mt-8;             /* margin-top: 32px; */
    }
  }
`} />
