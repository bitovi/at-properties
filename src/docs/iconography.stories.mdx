## Social

Social icons will be 40 x 40px, with 32px of space between icons, and 32px of space above the row of icons. Note the `:hover` state: the icon fill color changes to the social brand color.

Social "icons" are actually logos and must follow third-party brand guidelines like any other logo. Guidelines define not only the logo itself, but also it's color, size, placement, and spacing. Please reference each brand's fair use guidelines.

* **Facebook** - <a href="https://en.facebookbrand.com/facebookapp/">https://en.facebookbrand.com/facebookapp/</a>
* **Instagram** - <a href="https://en.instagram-brand.com/">https://en.instagram-brand.com/</a>

* **YouTube** - <a href="https://www.youtube.com/about/brand-resources/#logos-icons-colors">https://www.youtube.com/about/brand-resources/#logos-icons-colors</a>

## Accessibility considerations

Relevant <a href="https://www.w3.org/WAI/WCAG21/quickref/">WCAG 2.1 guidelines</a>:

* Same guidelines as for <a href="/?path=/story/foundation-color--page#accessibility-considerations">Color</a>.
* All non-text content that is presented to the user has a text alternative that serves the equivalent purpose. <a href="https://www.w3.org/WAI/WCAG21/quickref/#non-text-content">1.1.1 Non-text Content (Level A)</a>
* In content implemented using markup languages, the purpose of User Interface Components, icons, and regions can be programmatically determined. <a href="">1.3.6 Identify Purpose (Level AAA)</a>
