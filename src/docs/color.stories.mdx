# Color

The color palette includes utility, primary, secondary, neutral, and system colors.
* **Utility** colors include transparent, white, and black.
* **Primary** and **Neutral** colors include low and high contrast colors for text, borders, backgrounds.
* **Secondary** colors are tints and shades of the brand/accent color.
* **System** colors denote states or conditions such as error, alert, success, and interactive.

The DLP page will use a **subset** of the brand's <a href="https://www.figma.com/file/9Dh5lVMksIBNrDGMlPsWMk/DLP-Design-Library?node-id=39%3A34">full color palette</a>.

## How to configure

Define colors in **tailwind.config.js**. Tailwind will generate classes for `.text-{color}`, `.border-{color}`, and `.bg-{color}`.

<Source language='js' code={dedent`
    colors: {
        'transparent':  'transparent',
        'white':        '#FFFFFF',
        'black':        '#000000',
        primary: {
            100:        '#E7E9EB',
            200:        '#DBDDE1',
            500:        '#888F9C',
            900:        '#1B1D1F'
        },
        secondary: {
            500:        '#D22630'
        },
        neutral: {
            300:        '#B2B2B2',
            500:        '#808080'
        },
        system: {
            'focus':    '#0083CC'
        },
    },
`} />
