# Object Styles

Page elements can be styled with borders, rules/dividers, and accents.

## Borders

Borders are used as decorative accents. Borders vary in color and are single (1px) or double (4px). See individual components.

Components with a **single** 1px wide border:
* [Button](/?path=/docs/pattern-library-components-button--page) - 1px outer border
* [Figure](/?path=/docs/pattern-library-components-figure--page) - 1px inner border
* [Section Intro](/?path=/docs/pattern-library-containers-section-intro--page) - 1px inner border
* [Statistic](/?path=/docs/pattern-library-components-statistic--page) -- 1px inner border for [Company Stats](/?path=/docs/pattern-library-components-statistic--page#company-stats)

Components with a **double** (1px -- 2px space -- 1px) border:
* [Navigation](/?path=/docs/pattern-library-containers-navigation--page)
* [Section Content](/?path=/docs/pattern-library-containers-section-content--page)

## Rules

Rules are used to separate items. Rules are 1px wide, light gray (Primary 200). See individual components.

* [Section Content](/?path=/docs/pattern-library-containers-section-content--page) -- 1px divider between sections
* [Testimonial](/?path=/docs/pattern-library-components-testimonial--page) -- 1px divider between Testimonials
* [Property](/?path=/docs/pattern-library-components-property--default-story) -- 1px divider between Recent Sales
* [Statistic](/?path=/docs/pattern-library-components-statistic--page) -- 1px vertical and horizontal dividers between [Team Stats](/?path=/docs/pattern-library-components-statistic--page#team-stats)

## Accents

* Accents are red (Secondary 500), 2px high, 32 or 64px wide.
* Accents can appear above or below [headings](/?path=/story/pattern-library-components-heading--page) or [paragraphs](/?path=/story/pattern-library-components-paragraph--page), left aligned or centered.
* Spacing between accents and text is usually 16px. For `accent--below-center`, the spacing is text -- 16px -- accent -- 24px on mobile, 32px on desktop -- text. For level 6 headings (`.head-6`), the spacing is 8px -- accent -- 8px.

## How to configure

Define classes in **src/assets/object.styles.scss**.

<Source language='scss' code={dedent`
  /* Accent */
  .accent--above-left:before, 
  .accent--above-center:before, 
  .accent--below-center:after, 
  .accent--below-center-wide:after {
    @apply bg-secondary-500;   /* background-color: $secondary-500; */
    content: '';
    @apply block;              /* display: block; */
    height: 2px;
    @apply w-8;                /* width: 32px; */
    @apply mb-4;               /* margin-bottom: 16px; */
  }
  .accent--above-left:before {
    @apply ml-0;               /* margin-left: 0; */
    @apply mr-auto;            /* margin-right: auto; */
  }
  .accent--above-center:before {
    @apply ml-auto;            /* margin-left: auto; */
    @apply mr-auto;            /* margin-right: auto; */
  }
  .accent--below-center:after,
  .accent--below-center-wide:after {
    @apply mt-4;               /* margin-top: 16px; */
    @apply ml-auto;            /* margin-left: auto; */
    @apply mr-auto;            /* margin-right: auto; */
    @apply mb-6;               /* margin-bottom: 24px; */
    @media (min-width: theme('screens.lg')) {
      @apply mb-8;             /* margin-bottom: 32px; */
    }
  }
  .accent--below-center-wide:after {
    @apply w-16;               /* width: 64px; */
  }
  /* Accent: exceptions */
  .head-6.accent--above-center:before, 
  .head-6.accent--below-center:after {
    @apply my-2;               /* margin-top/bottom: 8px; */
    @apply mx-auto;            /* margin-left/right: auto; */
  }
  .accent--mb-0:after {
    @apply mb-0;               /* margin-bottom: 0px; */
  }
`} />

## Accessibility considerations

Relevant [WCAG 2.1 guidelines](https://www.w3.org/WAI/WCAG21/quickref/):

* [1.3.1 Info and Relationships (Level A)](https://www.w3.org/WAI/WCAG21/quickref/#info-and-relationships)<br />
  Information, structure, and relationships conveyed through presentation can be programmatically determined or are available in text. 
* [1.4.1 Use of Color (Level A)](https://www.w3.org/WAI/WCAG21/quickref/#use-of-color)<br />
  Color is not used as the only visual means of conveying information, indicating an action, prompting a response, or distinguishing a visual element.
