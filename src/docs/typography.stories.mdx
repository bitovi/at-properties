# Typography

## Font Family

Each brand (i.e., @properties, Ansley, Nest) has primary and secondary fonts and/or font weights/faces. 
* The **primary** font is used for body copy and some headings. It is the most prevalent font on screen.
* The **secondary** font is used as an accent, to bring more emphasis to some areas of the screen.

## Font Size

Tailwind CSS uses 10 font sizes ranging from `xs` to `6xl`. These are defined in **tailwind.config.js**.

<Source language='js' code={dedent`
  fontSize: {
    'xs':   '12px',
    'sm':   '14px',
    'base': '16px',
    'lg':   '20px',
    'xl':   '24px',
    '2xl':  '28px',
    '3xl':  '34px',
    '4xl':  '39px',
    '5xl':  '42px',
    '6xl':  '60px'
  },
`} />

* **Small** for screens less than 1280px wide
