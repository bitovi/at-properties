# Section

A section (coded as `article`) includes:
* [Section Intro]() -- optional animated header to introduce the section (e.g., Marketing, Selling, Partners)
* [Section Content]() -- content aligns to a `.content-grid`

See [Example: Selling](/?path=/docs/style-guide-structure--page#example-selling) on the [Structure](/?path=/docs/style-guide-structure--page) page.
