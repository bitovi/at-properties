# Animation

There are several areas of the page where animations play.
* **Title** area at the top of the page
* **Section intros** at the begining of some sections (e.g., Global Reach, Marketing, Community)
* **Transitions** added to select elements on a case by case basis (e.g., text fades in, images scale down)

## Transitions

* Transitions can be added to individual elements (e.g., paragraph, testimonial, property, etc.) using classes.
* Animations play once and do not reverse (rewind on scroll up) or repeat (start over once you reach the top of the page).
