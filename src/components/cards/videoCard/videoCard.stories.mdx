# Video Thumbnail

* A video thumbnail is a [Figure](/?path=/docs/pattern-library-components-figure--page) with an additional overlay for the play icon.
* Video thumbnails use a **16:9** [aspect ratio](/?path=/docs/style-guide-imagery--page#aspect-ratios).
* The play icon is 75% white by default, 100% white on `:focus` and `:hover`. 
* The play icon is centered vertically and horizontally; 48px wide on mobile/tablet; 128px wide on desktop.
* **Accessibility**: Notice that `.dlp-video-card` uses `aria-label="Watch video: ..."`. Read more about [writing alt text](/?path=/story/style-guide-imagery--page#writing-alt-text).
* **Animation**: All images scale down (shrink) on scroll down. Images can also fade in. Each `<figure>` can be given the class `.fade-in` (*not* indicated in code example below). See [Animation](/?path=/docs/style-guide-animation--page).
  * Shrink: Scale changes from 1.1 to 1 at the same speed as the user scrolls (scrub). 
  * Fade (optional): Opacity changes from 0 to 1 at the same speed as the user scrolls (scrub).

## How to configure

Define classes in **src/components/cards/videoCard/videoCard.styles.scss**.

<Source language='scss' code={dedent`
  /* Video Thumbnail */
  .dlp-video-card {
    @apply relative;           /* position: relative; */
    cursor: pointer;
    &:focus, &:hover {
      .dlp-video-card__overlay {
        @apply opacity-100;    /* opacity: 1; */
      }
    }
    .dlp-video-card__overlay {
      @apply absolute;         /* position: absolute; */
      @apply top-0;            /* top: 0; */
      @apply flex;             /* display: flex; */
      @apply items-center;     /* align-items: center; */
      @apply justify-center;   /* justify-content: center; */
      @apply w-full;           /* width: 100%; */
      @apply h-full;           /* height: 100%; */
      @apply opacity-75;       /* opacity: 0.75; */
      transition: opacity 0.3s;
    }
  }
  .icon-play {
    @apply w-12;               /* width: 48px; */
    @media (min-width: theme('screens.lg')) {
      @apply w-32;             /* width: 128px; */
    }
    @apply text-white;         /* color: theme('colors.white'); */
  }
`} />
