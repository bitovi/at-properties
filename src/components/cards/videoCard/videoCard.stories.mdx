# Video Thumbnail

* Video thumbnails use a **16:9** aspect ratio, have a border, and include a play icon.
* The play icon is 75% white by default, 100% white on `:focus` and `:hover`. 
* The play icon is 64px wide on mobile/tablet; 128px wide on desktop.

## How to configure

Define classes in **src/components/cards/videoCard/videoCard.styles.scss**.

<Source language='scss' code={dedent`
  /* Video Thumbnail */ 
  .video-thumbnail {
    @apply self-center;        /* align-self: center; */
    @apply text-white;         /* color: theme('colors.white'); */
    @apply justify-self-center;  /* justify-self: center; */
    @apply opacity-75;         /* opacity: 0.75; */
    &:focus, &:hover {
      @apply opacity-100;      /* opacity: 1; */
    }
  }
  .icon-play {
    @apply w-16;               /* width: 64px; */
    @media (min-width: theme('screens.lg')) {
      @apply w-32;             /* width: 128px; */
    }
  }
`} />
