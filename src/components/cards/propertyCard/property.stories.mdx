# Property

* A property includes an image (4:3 ratio), price, and address.
* Images will be JPG.
* The component uses a 2-col, 16px gap layout for mobile/tablet; 3-col = 1/3 + 2/3, 24px gap layout for desktop.
* **Spacing**: 
  * Properties are shown separated by a 1px light gray (Primary 100) horizontal rule, with 16px between the rule and each item.
  * 24/32px between the heading and properties (`.dlp-property-wrapper` top margin).
* **Accessibility**: Notice that images use `alt=""`. Read more about [writing alt text](/?path=/story/style-guide-imagery--page#writing-alt-text) for images. See [Figure](/?path=/story/pattern-library-components-figure--page).
* **Animation**: Properties fade in. Opacity changes from 0 to 1 at the same speed as the user scrolls (scrub). Each property is given the class `.fade-in`: `<figure class="dlp-property fade-in">` (*not* indicated in code example below). See [Animation](/?path=/docs/style-guide-animation--page).

## How to configure

Define classes in **src/components/cards/propertyCard/property.styles.scss**.

<Source language='scss' code={dedent`
  /* Property */
  .dlp-property-wrapper {
    @apply space-y-4;           /* 16px of space between items and dividers */
    @apply divide-y;            /* Place 1px horizontal dividers between items */
    @apply divide-primary-100;  /* Divider color is light gray (Primary 100) */
    @apply mt-6;                /* margin-top: 24px; */
    @media (min-width: theme('screens.lg')) {
      @apply mt-8;               /* margin-top: 32px; */
    }
  }
  .dlp-property {
    @apply grid;                 /* display: grid; */
    @apply grid-cols-2;          /* grid-template-columns: repeat(2, minmax(0, 1fr)); */
    @apply gap-4;                /* gap: 16px; */
    @apply pt-4;                 /* padding-top: 16px; */
    @media (min-width: theme('screens.lg')) {
      @apply grid-cols-3;        /* grid-template-columns: repeat(3, minmax(0, 1fr)); */
      @apply gap-6;              /* gap: 24px; */
    }
    &:first-child {
      @apply pt-0;               /* padding-top: 0; */
    }
  }
  .dlp-property__desc {
    @media (min-width: theme('screens.lg')) {
      @apply col-span-2;       /* grid-column: span 2 / span 2; */
    }
  }
  .dlp-property__price {
    @apply font-medium;        /* font-weight: 500; */
    @apply leading-none;       /* line-height: 1; */
  }
  .dlp-property__price + address {
    @apply mt-2;               /* margin-top: 8px; */
  }
`} />
