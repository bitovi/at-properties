# Social

* Icons are dark gray (Primary 900).
* Focus and hover states:
  * Facebook, Instagram, LinkedIn, Twitter, and YouTube change to their corporate color.
  * The earth icon (for web sites) changes to a dark blue.
  * Fair Housing and Realtor do not change color.
* **Spacing**: Padding around icons increases their total touch target size to 44 x 44px.
  * Mobile: icons are 24 x 24px + 10px padding (x2 sides = 20) = 44 x 44px touch target.<br />
    (icon 1) 10px padding + 4px right margin + 10px padding (icon 2) = 24px between icons.
  * Tablet/Desktop: icons are 32 x 32px + 6px padding (x2 sides = 12) = 44 x 44px touch target.<br />
    (icon 1) 6px padding + 20px right margin + 6px padding (icon 2) = 32px between icons.
  * No right margin on the last icon (`:last-child`).
  * 32px betweeen the list of icons and anything above it (top margin).
* **Animation**: The row of icons can fade in. Opacity changes from 0 to 1 at the same speed as the user scrolls (scrub). Each `<div class="dlp-social>` is given the class `.fade-in`: `<div class="dlp-social fade-in">` (*not* indicated in code example below). See [Animation](/?path=/docs/style-guide-animation--page).

## Brand guidelines

Social "icons" are actually logos and must follow third-party brand guidelines like any other logo. Guidelines define not only the logo itself, but also its color, size, placement, and spacing. Please reference each brand's fair use guidelines.

* **Facebook** - [https://en.facebookbrand.com/facebookapp/](https://en.facebookbrand.com/facebookapp/)
* **Instagram** - [https://en.instagram-brand.com/](https://en.instagram-brand.com/)
* **LinkedIn** - [https://brand.linkedin.com/](https://brand.linkedin.com/)
* **Twitter** - [https://about.twitter.com/en_us/company/brand-resources.html](https://about.twitter.com/en_us/company/brand-resources.html)
* **YouTube** - [https://www.youtube.com/about/brand-resources/#logos-icons-colors](https://www.youtube.com/about/brand-resources/#logos-icons-colors)
* **HUD Equal Housing** - [https://www.hud.gov/library/bookshelf11/hudgraphics](https://www.hud.gov/library/bookshelf11/hudgraphics)
* **National Association of Realtors** - [https://www.nar.realtor/logos-and-trademark-rules/the-realtor-logo](https://www.nar.realtor/logos-and-trademark-rules/the-realtor-logo)
