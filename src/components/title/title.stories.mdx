# Title

* As soon as the page loads, the animation starts. <span className="highlight">Include a delay? Check that the bg image and SVGs have loaded?</span>
* The background image scales down from 150% to 100% over 4.5 seconds (timeline 1).
* Meanwhile (timeline 2), after a delay of 700ms, 
* The border fades in over 700ms.
* The @properties logo fades in and slides up over 700ms.
* The word "love" (or an alternative SVG) writes itself (each letter has its own timing).
* The "presented by" text fades in and slides up over 700ms.
* The "scroll to continue" text fades in and slides up over 700ms.
* The arrow fades in and slides *down* over 700ms.
* The arrow moves up and down on an infinite loop. <span className="highlight">Should we change this to 5X instead?</span>
* Scrolling is disabled while the animation is running. All subsequent sections are hidden using `display: none`. 
* Once the animation finishes, scrolling is enabled. All subsequent sections are made visible using `display: block`.

<span className="highlight">[ Include a visual here? Movie, Canvas, Preview? ]</span>

There are two styles for the Title: `.title--style-light` and `.title--style-dark`.
* **Light** has a light background and dark gray (Primary 900) text and graphics (SVGs)
* **Dark** has a dark background and white text and graphics (SVGs)

## How to configure

Define classes in **src/components/title/title.styles.scss**.

<Source language='scss' code={dedent`
  /* Title */ 
  #love {
    @apply block;              /* display: block; */
    @apply m-auto;             /* margin: auto; */
    @apply w-1/2;              /* width: 50%; */
    @media (min-width: theme('screens.md')) {
      @apply w-1/3;            /* width: 33.333333%; */
    }
    @media (min-width: theme('screens.lg')) {
      @apply w-1/5;            /* width: 20%; */
    }
  }
  p.scroll {
    @apply text-xs;            /* font-size: theme('fontSize.xs'); */
    @apply font-medium;        /* font-weight: 500; */
    @apply tracking-widest; 	 /* letter-spacing: 0.1em; */
    @apply uppercase;          /* text-transform: uppercase; */
  }
  /* Title: styles */
  .title--style-dark {
    @apply bg-gradient-to-t;     /* background: linear-gradient(to top, */
    @apply from-black;           /* --gradient-from-color: theme('color.black'); */
    @apply to-primary-900;       /* --gradient-to-color: theme('color.primary.900'); */
    h1, h2, h3, h4, h5, h6, p {
